# LeiAI-WEB-H5


### 最新版本： leiai-v7.0.0(20240512发布)

### 功能介绍
- 对话问答，图片生成等，均支持高并发账号池
- 大模型驱动的应用，支持三方应用接入
- 大模型支持被动同步国内外最新版本，无需更新系统
- 独立后台管理，包括系统配置和运营管理等
